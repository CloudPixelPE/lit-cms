# OrisonJS Netlify Starter Kit

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/megazear7/orison-netlify-starter-kit)

## Install

```
npm install
```

## Build

```
npm run build
```

## Develop

```
npm run serve
```
